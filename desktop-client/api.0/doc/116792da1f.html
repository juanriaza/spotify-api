observe

Register an observer. The Playlist object fires the following events:
  • EVENT.LOAD: Called when the playlist has completed loading.
  • EVENT.UNLOAD: Called when the playlist is unloaded from memory.
  • EVENT.CHANGE: Called when the playlist is changed.
  • EVENT.ITEMS_ADDED: Called when tracks have been added to the playlist.
  • EVENT.ITEMS_REMOVED: Called when tracks have been removed from the playlist.
  • EVENT.ITEMS_MOVED: Called when tracks have been rearranged within the playlist.
  • EVENT.RENAME: Called when the playlist is renamed.
Source:
  • models.js, line 1405
Example
var sp = getSpotifyApi();
var models = sp.require('$api/models');

var playlist = models.Playlist.fromURI('spotify:user:spotify:playlist:3Yrvm5lBgnhzTYTXx2l55x');
playlist.observe(models.EVENT.RENAME, function() {
  console.log('Playlist renamed!');
});
