observe

Register an observer. The Session object fires the following events:
  • EVENT.LOGIN: Fires when the user logs into the Spotify client. Will not be fired if your application isn't active when this happens.
  • EVENT.LOGOUT: Fires when the user logs out from Spotify client. Will not be fired if your application isn't active when this happens.
  • EVENT.STATECHANGED: Fired when the state property changes.

Example
var sp = getSpotifyApi();
var models = sp.require('$api/models');

models.session.observe(models.EVENT.STATECHANGED, function() {
  console.log('Session state changed!');
});
