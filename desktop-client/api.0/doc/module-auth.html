<inner> authenticateWithFacebook

Helper method for authenticating against Facebook Connect using showAuthenticationDialog.
Parameters:
Name Type Description
appId string Your Facebook application ID.
permissions Array.<string> An array of permission keys, as documented at http://developers.facebook.com/docs/reference/api/permissions/.
callbacks Object An object containing the following callback functions:
  • onSuccess(accessToken, ttl)
  • onFailure(error)
  • onComplete().

Example
var sp = getSpotifyApi();
var auth = sp.require('$api/auth');

auth.authenticateWithFacebook('MY_APP_ID', ['user_about_me', 'user_checkins'], {
  onSuccess: function(accessToken, ttl) {
    console.log('Success! Access token: ' + accessToken);
  },
  onFailure: function(error) {
    console.log('Authentication failed with error: ' + error);
  },
  onComplete: function() {}
});

<inner> showAuthenticationDialog

Shows a popup window which will render the contents of initialUrl. The popup will stay on screen until the user closes it, or a new resource is loaded that has successURL as its prefix or complete path. Use this function to authenticate against Facebook, etc.
Parameters:
Name Type Description
initialUrl string The initial URL to load.
successUrl string The URL that will be loaded upon successful login.
callbacks Object An object containing the following callback functions:
  • onSuccess(response)
  • onFailure(error)
  • onComplete().
Source:
  • auth.js, line 43
Example
var sp = getSpotifyApi();
var auth = sp.require('$api/auth');

auth.showAuthenticationDialog('http://www.last.fm/api/auth/?api_key=LAST_FM_API_KEY&cb=sp://my_app_name', 'sp://my_app_name', {
  onSuccess: function(response) {
    // Response will be something like "sp://my_app_name?token=xxxxxxx"
    console.log('Success! Response URL: ' + response);
  },
  onFailure: function(error) {
    console.log('Authentication failed with error: ' + error);
  },
  onComplete: function() {}
});
