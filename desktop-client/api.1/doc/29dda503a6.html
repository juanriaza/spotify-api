addEventListener

Same as the addEventListener in standard JavaScript. Call this method to add an observer to the receiver object, for a given named event.
Parameters:
Name Type Description
eventType string The name of the event to get notifications for.
observer Function The callback function. To unregister the callback, the exact same function must be passed to the removeEventListener method.
Since:
  • 1.0.0
Source:
  • models.js, line 247
See:
  • module:api/models~Observable#removeEventListener
Example
models.player.addEventListener('change', updateNowPlayingWidget);

dispatchEvent

Triggers an event for all observer callbacks. Any callback that explicitly returns false or calls preventDefault() will cause the return value of this function to be false.
Parameters:
Name Type Description
evt {type: string} | string The event object to dispatch (must have a type property), or simply the event type as a string.
Since:
  • 1.0.0
Source:
  • models.js, line 304
Returns:
False if the event was prevented; otherwise, true.
Type
boolean

removeEventListener

Same as the removeEventListener in standard JavaScript. Call this method to remove an observer that was previously added.
Parameters:
Name Type Description
eventType string The name of the event you're getting notifications for.
observer Function The callback must be the exact same function instance as was used in the registration, and not just an identical anonymous function.
