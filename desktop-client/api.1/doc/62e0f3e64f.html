Class: AnalyticsContext

api/core~SpotifyApi. AnalyticsContext

An analytics context can be used to group activity inside of an application so it can be tracked as a single logical entity. All API requests executed when a given context is active will be tagged with the context id so they can be assigned to the context on the other side of the bridge. AnalyticsContexts can handle asyncronous callbacks and may be nested arbitrarily deep.

new AnalyticsContext

Never construct an AnalyticsContext using the constructor - use the SP.analyticsContext() convenience function instead.
