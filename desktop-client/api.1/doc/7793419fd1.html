Class: Callback

api/core~SpotifyApi. Callback

A callback class that captures a context stack on instantiation and restores it on invocation.

new Callback

Never construct a Callback using the constructor - use the SP.callback() convenience function instead.
Parameters:
Name Type Description
func Function The function to wrap.
opt_contextStack Array= A context stack to capture. If a stack is not provided, this defaults to the current context stack.
