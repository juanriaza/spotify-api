showContextUI

Show the platform-specific context UI.
Parameters:
Name Type Description
item Array.<Track | Album | Artist | Playlist> | Track | Album | Artist | Playlist Items to show context UI for. The items in the array should be of the same type.
opt_point {x: number, y: number}= Point at which the context UI should be rendered. This point is in x,y coordinates relative to the application viewport.
opt_origin Album | Artist | Playlist= Origin where the items come from.
Source:
  • models.js, line 1427
Returns:
A Promise which will resolve successfully if the context UI is opened, and which will fail if the context UI could not be opened.
Type
Promise
Example
var items = [
  models.Track.fromURI('spotify:track:2e2Z8FeqqvUClWqc23nuX1'),
  models.Track.fromURI('spotify:track:3k68IqyXiefjfjKy3BVOX0'),
  models.Track.fromURI('spotify:track:3jRHAsjvnSTmB5crrpqyTj')
];
var element = document.getElementById('track');
var rect = element.getBoundingClientRect();
models.client.showContextUI(items, { x: rect.left, y: rect.top })
    .done(contextUIOpenedSuccessfully)
    .fail(contextUINotOpened);
