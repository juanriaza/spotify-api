Location

An object containing location information for the current session.

new Location

Properties:
Name Type Description
accuracy number The accuracy level of the latitude and longitude coordinates in meters.
latitude number Geographic latitude, specified in decimal degrees.
longitude number Geographic longitude, specified in decimal degrees.
