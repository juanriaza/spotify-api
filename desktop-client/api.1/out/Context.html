Class: Context

Context

An opaque context. The player will hold one of these contexts when a track is playing, rather than holding a reference to the high level context objects, such as Album or Playlist. This context can be compared to other contexts to check for equality and it can also be passed to the play methods of the player object.

new Context

Never construct a context object using the default constructor - use fromURI() instead.

fromURI

Returns an opaque context for a given Spotify URI. The URI is the only property that is available on the cotext. It is loaded immediately.
