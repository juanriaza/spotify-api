<static> forItem

Factory method.

Creates quick action buttons for a specific item.
Parameters:
Name Type Description
item Track | Album | Artist An item.
opt_origin Object= Origin for the request. For example, if it comes from a list showing a playlist, the playlist uri will be the origin. Can be used to let the widget know about the context.
opt_indexGetter Function= Optional index getter method, which is used to get an index to send to the Context Menu.
