<inner> options

These are the available options for the Scroll Agent.
Properties:
Name Type Description
id String The string id, used for debugging purposes. Defaults to an unique encoded string.
container HTMLElement The nodes container. Defaults to the view element.
hotZone Number The additional area of the screen used to render items. Defaults to 0.
deadZone Number The amount of pixels to scroll to trigger a new load. Defaults to 0.
debug Boolean Prints debug messages to the console. Defaults to false.
removeInvisibles Boolean Removes invisible nodes from the dom. Defaults to false.
length Number The total length of nodes.
tagName String The tag name of the spacing elements. Defaults to 'div'.
height Number The default height of the spacing elements. Defaults to 0.
limitDelta Number If set, limits the scrolling rate to the specified number. Defaults to false.
batchSize Number The amount of nodes to request.
