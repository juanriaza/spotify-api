Class: CustomButton

views/buttons~ CustomButton

A button without any appearance, optionally with a label. The custom button must by styled with regular CSS by adding custom CSS classes to the node. It is better to use the this button as a base for a customized button rather than a regular HTML button element, because they will not track well on all platforms, such as iOS and Android. To create an instance of this button, call the withClass factory method.

new CustomButton

Use a factory method to instantiate.

setAccentuated

Set the accentuation of the button. An accentuated button will appear in a way such as to draw the attention of the user. In the current version of Spotify the button will have a green tint. This might change in the future.
Parameters:

opt_effect string= The effect of the button. Not considered if the first argument is false. 'positive': Use a positive color (green in this version). Default. 'negative': Use a negative color (red in this version).
