Class: CustomButton

CustomButton

new CustomButton

A button without any appearance, optionally with a label. The custom button must by styled with regular CSS by adding custom CSS classes to the node. It is better to use the this button as a base for a customized button rather than a regular HTML button element, because they will not track well on all platforms, such as iOS and Android. To create an instance of this button, call the withClass factory method.
