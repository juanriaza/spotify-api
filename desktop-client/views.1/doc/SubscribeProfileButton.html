Class: SubscribeProfileButton

SubscribeProfileButton

new SubscribeProfileButton

A subclass of SubscribeButton that knows how to subscribe to artists and users. It will properly update itself in response to the artist or user it is targeting being subscribed to/unsubscribed from somewhere else.
Parameters:
Name Type Description
profile Artist | User The artist or user to subscribe to when the button is clicked.
