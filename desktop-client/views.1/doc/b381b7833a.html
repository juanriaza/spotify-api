addToDom

Add the tab bar to the DOM.
Parameters:
Name Type Description
element HTMLElement Reference element for insertion point.
position string Where to add it, relative to the reference element. Valid values:
  • 'append': Add tab bar after the last element inside the reference element. If reference is empty, it will be the first child.
  • 'prepend': Add tab bar before the first element inside the reference element. If reference is empty, it will be the first child.
  • 'after': Add tab bar after the reference element.
  • 'before': Add tab bar before the reference element.
Since:
  • 1.22.0
Source:
  • tabbar.js, line 276
Example
tabBar.addToDom(document.body, 'prepend');

<static> withTabs

Factory method.

Returns a tab bar with the specified tabs.
Parameters:
Name Type Description
tabs Array.<Object> An array of tab definitions.
opt_options Object= An object with options. These are available:
  • secondClickScrolls: Whether to scroll to the top when a tab is clicked the second time. First click activates tab, second scrolls to top.
  • sticky: Whether to stick to the top when scrolled past. Default: true.
Since:
  • 1.22.0
Source:
  • tabbar.js, line 120
Returns:
A new tab bar.
Type
TabBar
Example
require(['$views/tabbar#TabBar'], function(TabBar) {

  var tabBar = TabBar.withTabs([
    {id: 'overview', name: 'Overview', active: true},
    {id: 'followers', name: 'Followers'},
    {id: 'following', name: 'Following'}
  ]);
  tabBar.addToDom(document.body, 'prepend');

  // Use this event to know when to change content based on the tab
  tabBar.addEventListener('tabchange', function(e) {
    e.id; // ID for the tab it changed to
    e.previousId; // ID for the tab that was active before
  });

  // Use this event to be able to preload content before the user switched to the tab
  tabBar.addEventListener('tabhover', function(e) {
    e.id; // ID for the tab that the cursor hovers over
    e.active; // Whether the tab is currently active
  });

});
